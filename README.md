# website-highlight-saver

A website highlight saver chrome extension. To use it:
- Open Extension/ Settings: In your Chrome address bar, navigate to chrome://extensions/.
- Enable Developer Mode: In the top-right corner of the Extensions page, toggle the "Developer mode" switch to ON.
- Load Unpacked: Click the "Load unpacked" button in the top-left.
- Select Folder: Select the project folder: `<downloaded-path>/website-highlight-saver`
- Pin the Extension: Click the jigsaw puzzle piece icon in Chrome's toolbar and click the Pin icon next to "Website Highlight Saver" so it stays visible.
